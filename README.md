# ZERODHA Clone

A full-stack Zerodha trading platform clone built using the MERN stack. This project replicates core functionalities of the Zerodha trading platform with a modern and responsive user interface.

# Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Bootstrap / CSS

# Features

* User Authentication
* Dashboard
* Buy/Sell Interface
* Holdings Section
* Responsive UI

# Installation

git clone https://github.com/pkmallick/ZERODHA-_Clone.git
cd ZERODHA-_Clone
npm install
npm start
