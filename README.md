# 📈 Zerodha Clone - Full Stack Trading Platform

A full-stack trading platform inspired by **Zerodha**, built using the MERN stack. This project demonstrates user authentication, stock portfolio management, order handling, and a responsive trading dashboard.

# ⚙️ Installation & Setup

## 1. Clone Repository

```bash
git clone:https://github.com/pkmallick/ZERODHA-_Clone.git
```

Navigate into the project:

```bash
cd ZERODHA-CLONE
```

---

# Backend Setup

Go to backend:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```
PORT=3002
MONGO_URL=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Start backend:

```bash
npm run dev
```

Backend runs on:

```
http://localhost:3002
```

---

# Frontend Setup

Open another terminal:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
npm start
```

Frontend runs on:

```
http://localhost:3000
```

---

# Dashboard Setup

Open another terminal:

```bash
cd dashboard
```

Install dependencies:

```bash
npm install
```

Start dashboard:

```bash
npm start
```

Dashboard runs on:

```
http://localhost:3001
```
